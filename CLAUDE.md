# Project Rules

This project is a modern website for a pizzeria.

## Tech stack
- React / Next.js (App Router, TypeScript)
- Tailwind CSS
- Mobile-first design
- Clean component structure

## Design style
- Modern, premium, simple
- Strong hero section
- Clear CTA buttons
- Good spacing
- No generic AI-looking layout
- No random gradients unless useful

## Code rules
- Use reusable components
- Keep code readable
- Avoid unnecessary dependencies
- Make all pages responsive
- Optimize images
- Test layout on mobile

## Business goal
This is a website for a pizzeria, designed primarily to display the menu and allow customers to quickly reserve a table without having to call, with the option to order just pizzas as well.
